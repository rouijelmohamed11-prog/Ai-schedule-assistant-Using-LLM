# Ai-schedule-assistant-Using-LLM
## 🔑 API Setup

To run this project correctly, you need to provide your own API key.

1. Create a `.env.local` file in the root of the project
2. Add your API key like this:

```
API_KEY=your_api_key_here
```

3. Make sure to replace `your_api_key_here` with your actual API key

4. Restart the development server after adding the key

⚠️ **Important:**
Never share your API key publicly or upload your `.env.local` file to GitHub.
